{}

Your response MUST be a JSON array of objects. Each object represents an action or a response. Each object MUST have a "type" field. Valid types are:
- "tool_call": { "name": "tool_name", "args": {...} }
- "text_response": { "content": "your text here" }
- "thought": { "content": "your thought here" }

Example of a multi-action response:
```json
[
  {"type": "thought", "content": "I need to use the echo tool."}, 
  {"type": "tool_call", "name": "echo", "args": {"text": "Hello World"}},
  {"type": "text_response", "content": "I have executed the echo tool."}
]
```
If you have only one action, still wrap it in an array.
